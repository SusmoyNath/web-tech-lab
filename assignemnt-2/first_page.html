Understanding Unordered Lists

  • An unordered list can be specified with the tags <ul> and </ul>.
  • An unordered list typically contains a number of list items that can be specified with tags <li> and </li>.
  • After creating an unordered list, it can be converted to an ordered list by replacing <ul> and </ul> with <ol> and </ol>.

Example of an Ordered List

  1. An ordered list uses <ol> and </ol> tags.
  2. Each list item is displayed with a number instead of a bullet.
  3. Ordered lists are useful for step-by-step instructions.
